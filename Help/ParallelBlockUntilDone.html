About the Parallel Block Until Done Tool:

The Parallel Block Until Done Tool will write all input records from Input1 to Output1 and all input records from Input2 to Output2. As with the standard block until done tool it stops downstream processing until the very last record has arrived for each input/output pair. Further it writes all records first to output1 before then starting writing all records to output2.

Configuration Properties:

There is no configuration properties to the tool.
